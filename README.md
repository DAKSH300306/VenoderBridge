# VendorBridge ERP

## Overview

VendorBridge ERP is a Procurement and Vendor Management System developed for the Odoo Hackathon.

## Features

* Vendor Management
* RFQ Management
* Quotation Management
* Quotation Comparison
* Approval Workflow
* Purchase Order Generation
* Invoice Generation

## Technology Stack

* Python
* Flask
* SQLite
* Bootstrap 5

## Workflow

Vendor → RFQ → Quotation → Approval → Purchase Order → Invoice
